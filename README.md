Moonveil — Where Every Scent Tells a Story 

A modern, fully responsive perfume brand UI crafted with React and Tailwind CSS v4.
Clean design, smooth navigation, and elegant layout — built for the web.

## Features

- Responsive Navbar with smooth scroll
- Hero section with background image
- About section
- Perfume collection grid
- Footer with links
